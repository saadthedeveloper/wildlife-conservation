# Wildlife Conservation Society Website

A **responsive, interactive, and visually engaging website** for the Wildlife Conservation Society (WCS), designed to promote wildlife conservation, showcase initiatives, and encourage public support through donations and memberships.

---

## Features

* **Responsive Layout:** Works on desktop and mobile devices.
* **Home Section:** Hero images with layered hills, trees, and wildlife visuals.
* **About Us:** Mission, vision, and global conservation initiatives.
* **Locations:** Interactive map with WCS locations using **Leaflet.js**.
* **Careers:** Highlighting opportunities and reasons to join WCS.
* **Newsroom:** Latest news and editor’s picks with image cards.
* **Donation Section:** Predefined amounts and custom donation option.
* **Custom Cursor & Interactive Elements:** Enhanced user experience.
* **Footer:** Social links, contact info, latest news, and important links.

---

## Technologies

* **HTML5** – Page structure
* **CSS3** – Styling and responsive design
* **JavaScript** – Interactive features (`script.js`)
* **Leaflet.js** – Interactive maps (`map.js`)
* **Google Fonts & Font Awesome** – Typography and icons

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/saadthedeveloper/WCS-Website.git
   ```
2. Navigate to the project folder:

   ```bash
   cd WCS-Website
   ```
3. Open `index.html` in your browser.

*No backend is required; this is a static website.*

---

## Usage

* Scroll through the site to explore WCS content.
* Click **Donate** or **Membership** buttons to redirect to donation/membership pages (optional).
* Use the **Locations** section to view the interactive map.
* Explore career opportunities and latest news.
